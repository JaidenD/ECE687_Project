# multi_robomaster_ros_sim
ROS2 package to simulate a system of DJI RoboMaster robots

The `run.sh` script will start a Docker container from the image `dji_robomaster_ros:1.0` and launch the simulator node.

The image `dji_robomaster_ros:1.0` can be built following the instructions in the `setup` folder of the repository `https://github.com/erablab/multi_robomaster_ros`.
